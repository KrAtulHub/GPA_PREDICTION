# 🎓 Student GPA Prediction App

A **Machine Learning Web Application** that predicts a student's **Grade Point Average (GPA)** based on academic behavior, demographics, and extracurricular activities.

The application is built using **Python, Machine Learning, and Streamlit**, allowing users to input student data and instantly receive a GPA prediction.

# 📊 Project Overview

Student performance depends on many factors such as:

* Study habits
* Absences
* Parental support
* Extracurricular activities
* Tutoring
* Demographics

This project uses **Machine Learning (KNN Regression)** to analyze these factors and predict a student's GPA.

---

# 🧠 Machine Learning Workflow

1️⃣ Data Collection
2️⃣ Data Preprocessing
3️⃣ Feature Selection
4️⃣ Train-Test Split
5️⃣ Feature Scaling using StandardScaler
6️⃣ Model Training using **KNN Regressor**
7️⃣ Model Evaluation
8️⃣ Deployment using **Streamlit**

# ⚙️ Run Locally

Clone the repository:

```
git clone https://github.com/KrAtulHub/GPA_PREDICTION.git
```

Navigate to the project folder:

```
cd GPA_PREDICTION
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
streamlit run app.py
```
